### 기본 정보
- **id**: "dice-grid-tactics"
- **title**: "🎲 다이스 그리드: 택틱스"
- **description**: "보드 위에 주사위를 배치하여 싸우는 턴제 전략 퍼즐 로그라이트"
- **loader**: "() => import('./games/dice-grid-tactics/main.js')"

### 레퍼런스 게임
- **원작**: Die in the Dungeon (2021, Dicey Bird)
- **장르**: 주사위 덱빌딩 / 턴제 전략
- **원작의 핵심**: 주사위를 카드로 취급하고, 슬롯에 배치해 시너지를 일으키는 직관적인 룰과 깊이 있는 빌드 다양성으로 큰 인기를 끌었습니다.

### 혁신 트위스트
> 적과 내가 한 판 위에서 주사위 배치를 조율하는 실시간 공간적 공방전
- **트위스트 유형**: 시점/컨트롤 반전 + 자원 딜레마
- **핵심 변경점**: 단순히 내 슬롯에 주사위를 놓는 것이 아니라, **적과 내가 하나의 5x5 공유 보드 위에서 주사위를 배치**하며 싸웁니다. 적 주사위와 인접하게 배치하면 내 주사위가 강화되거나 약화되고, 적은 내 주사위의 눈을 강제로 돌려놓거나(Flip) 밀쳐낼(Push) 수 있습니다.
- **플레이어 경험**: 언제 주사위를 굴려 위험을 감수할지, 적 주사위의 행동 범위를 어떻게 차단하고 보드를 장악할지 영리한 위치 공방전을 경험하게 됩니다.

### 핵심 게임루프
```
주사위 굴리기 → 공유 보드에 배치 → 배치 격자 효과 및 시너지 발동 → 보드 정리 및 보상 선택 → 다음 턴/스테이지
```

### 픽셀아트 비주얼 가이드
- **해상도**: 16x16 주사위 및 몬스터 스프라이트
- **팔레트**: 16색 던전 팔레트 (어둡고 차분한 브라운/그레이 + 네온 퍼플 주사위 포인트)
- **애니메이션**: 주사위 롤링 (4프레임), 캐릭터 대기 (2프레임), 피격 (2프레임)
- **배경 스타일**: 5x5 체스판 스타일의 심플한 네온 격자 타일맵

### Phaser JS 구현 포인트
- **Physics**: 필요 없음 (Grid 기반 좌표 계산만 사용하므로 무거운 물리 엔진을 비활성화하여 리소스 절약)
- **Scene 구조**: Boot → Preload → Menu → BattleScene → UpgradeUI
- **핵심 시스템**: `DiceManager` (주사위 드래그, 눈 관리), `GridBoard` (5x5 보드 상태 및 시너지 판정), `EnemyAI` (점수 기반 보드 배치 알고리즘)
- **재사용 모듈**: `Card/ItemDragComponent` (마우스 드래그 & 보드 스냅 기능)
- **난이도**: ⭐⭐⭐ (그리드 판정 및 주사위 드래그 앤 드롭 구현 중심)

### 시장성 평가
- **타겟 유저**: 턴제 전략 및 퍼즐 게임을 좋아하는 하드코어/캐주얼 게이머
- **세션 길이**: 5~10분 (한 판당 빠른 호흡)
- **재플레이성**: 높음 (다양한 눈을 가진 주사위 시너지와 무작위 릴렉트 보상 시스템)
- **참고 성공 사례**: Die in the Dungeon, Dicey Dungeons, Into the Breach

## 아이디어 2: 👾 픽셀 로스: 소실된 입자들

### 기본 정보
- **id**: "pixel-loss"
- **title**: "👾 픽셀 로스: 소실된 입자들"
- **description**: "행동할 때마다 몸의 픽셀이 소실되어 크기가 작아지는 퍼즐 플랫포머"
- **loader**: "() => import('./games/pixel-loss/main.js')"

### 레퍼런스 게임
- **원작**: Celeste (2018, Extremely OK Games)
- **장르**: 퍼즐 플랫포머
- **원작의 핵심**: 정밀한 조작감과 대시(Dash)를 이용한 레벨 디자인으로, 죽어도 즉시 부활하는 기분 좋은 도전 욕구를 자극하여 대히트했습니다.

### 혁신 트위스트
> 캐릭터가 커스텀 '액션 자원'으로서 자신의 몸체를 직접 깎아가며 장애물을 통과하는 기믹
- **트위스트 유형**: 자원 딜레마 (Resource Dilemma)
- **핵심 변경점**: 플레이어의 몸은 32x32 크기의 픽셀 덩어리입니다. **점프, 대시를 할 때마다 몸을 구성하는 픽셀(질량)이 떨어져 나갑니다.** 픽셀이 많이 떨어질수록 크기가 작아져 좁은 틈을 지나갈 수 있게 되지만, 점프 높이가 낮아지고 피격 시 즉사하기 쉬워집니다. 다시 원래 크기로 돌아가려면 스테이지 곳곳에 흩어진 자신의 '소실된 픽셀'을 다시 흡수해야 합니다.
- **플레이어 경험**: 좁은 틈을 지나가기 위해 일부러 대시를 연타하여 몸을 깎아낼지, 아니면 안전하게 질량을 보존하여 높은 곳에 올라갈지 등 크기 변화와 능력의 트레이드오프를 고민하는 퍼즐을 마주하게 됩니다.

### 핵심 게임루프
```
이동 및 행동(픽셀 소실) → 크기 축소 → 퍼즐/장애물 통과 → 픽셀 회수(크기 회복) → 스테이지 탈출
```

### 픽셀아트 비주얼 가이드
- **해상도**: 32x32 플레이어 캐릭터 (소실 단계에 따라 24x24, 16x16, 8x8로 실시간 변화), 16x16 타일맵
- **팔레트**: PICO-8 팔레트 (16색의 비비드하고 레트로한 감성)
- **애니메이션**: 걷기 (4프레임), 점프/체공 (2프레임), 픽셀 비산 효과 (입자 파티클)
- **배경 스타일**: 네온 컬러가 강조된 미니멀한 지하 연구소 타일맵

### Phaser JS 구현 포인트
- **Physics**: Arcade Physics (정밀하고 가벼운 플랫폼 물리 지원, `body.setSize()`로 물리 히트박스 실시간 축소 조정)
- **Scene 구조**: Boot → Preload → MainMenu → LevelSelect → GameplayScene
- **핵심 시스템**: `SizeStateManager` (캐릭터 소실 상태 관리 및 렌더링 스케일링), `ParticleEmitter` (소실 시 흩어지는 픽셀 파티클 시스템)
- **재사용 모듈**: `TilemapPlatformCollision` (타일맵 충돌 기본 템플릿)
- **난이도**: ⭐⭐⭐⭐ (히트박스 유동적 제어 및 입자 흡수 물리 연산 필요)

### 시장성 평가
- **타겟 유저**: 독특한 기믹의 플랫포머 및 퍼즐 매니아
- **세션 길이**: 스테이지당 1~3분 (전체 세션 10~15분)
- **재플레이성**: 중간 (고유한 스피드런 및 소실 최소화 도전과제)
- **참고 성공 사례**: VVVVVV, Celeste, Thomas Was Alone

## 아이디어 3: 🔌 스웜 해킹: 보안 시스템 침투

### 기본 정보
- **id**: "swarm-hacking"
- **title**: "🔌 스웜 해킹: 보안 시스템 침투"
- **description**: "보안 프로그램의 타워를 뚫고 코어를 감염시키는 리버스 타워 디펜스"
- **loader**: "() => import('./games/swarm-hacking/main.js')"

### 레퍼런스 게임
- **원작**: Kingdom Rush (2011, Ironhide Game Studio)
- **장르**: 리버스 타워 디펜스 / 실시간 전략
- **원작의 핵심**: 다양한 타워 배치와 업그레이드로 밀려오는 적을 막아내는 디펜스 게임의 기본에 충실한 재미를 주어 전 세계적으로 유행했습니다.

### 혁신 트위스트
> 방어가 아닌 공격자의 입장에서 다수의 무리 지능과 디버깅 스킬을 활용하여 통제된 격자망을 뚫는 재미
- **트위스트 유형**: 시점/컨트롤 반전 (Perspective Inversion) + 장르 크로스오버
- **핵심 변경점**: 방어 타워를 설치하는 것이 아니라, **방화벽 타워가 빽빽한 시스템 내부로 침투하는 바이러스 무리의 지휘관(Swarm Commander)**이 됩니다. 실시간으로 다양한 면역 저항력을 가진 바이러스 패킷들을 조합하여 방화벽의 화망을 뚫어야 합니다. 특정 타워를 해킹해 아군을 돕거나, 경로를 실시간으로 우회하는 기능이 추가됩니다.
- **플레이어 경험**: 적의 방어 타워 공격 속도와 범위를 파악하고, 아군 유닛들의 쉴드 타이밍을 실시간으로 발동하여 강력한 보안망을 뚫고 지나가는 짜릿한 돌파의 쾌감을 느끼게 됩니다.

### 핵심 게임루프
```
시스템 노드 분석 → 최적 바이러스 패킷 조합/배출 → 실시간 스킬 발동(해킹/우회) → 코어 감염 및 자원 획득 → 패킷 업그레이드
```

### 픽셀아트 비주얼 가이드
- **해상도**: 16x16 소형 바이러스 및 타워 스프라이트
- **팔레트**: Dark Cyberpunk (검은색 배경 + 밝은 사이언 Blue, 그린 Green, 네온 Orange 포인터)
- **애니메이션**: 바이러스 패킷 흐름 (2프레임 루프), 타워 발사 (3프레임), 코어 폭발 (6프레임)
- **배경 스타일**: 인쇄 회로 기판(PCB) 또는 매트릭스 그리드 스타일의 사이버 디렉토리 맵

### Phaser JS 구현 포인트
- **Physics**: Arcade Physics (속도 및 유닛 경로 추적을 위한 가벼운 물리 연산)
- **Scene 구조**: Boot → Preload → TerminalMenu → LevelSelect → CyberNetScene
- **핵심 시스템**: `PathFollower` (Phaser Path API를 활용한 무리 이동 시스템), `HackingAbilityManager` (쿨다운 기반 액티브 해킹 스킬), `TowerDefenseAI` (자동 타깃팅 공격 타워 오브젝트 풀)
- **재사용 모듈**: `PathCreator` (Phaser의 Curve/Path 시스템을 활용한 적/아군 이동 경로 설정기)
- **난이도**: ⭐⭐⭐⭐⭐ (다수의 유닛 이동 제어 및 타워 투사체 물리 처리로 난이도 높음)

### 시장성 평가
- **타겟 유저**: 사이버펑크 감성을 좋아하는 RTS 및 디펜스 게이머
- **세션 길이**: 노드당 3~5분
- **재플레이성**: 높음 (침투 유닛 빌드 트리 커스터마이징 및 맵별 최적 경로 설계)
- **참고 성공 사례**: Anomaly 시리즈, Creeper World, MIND CONTROL DELETE
