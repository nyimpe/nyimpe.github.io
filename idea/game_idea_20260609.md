
## 🎮 주간 게임 아이디어

### 아이디어 1: 픽셀 던전 크롤러 (Pixel Dungeon Crawler)

*   **게임 제목:** 에테르 던전: 픽셀의 유산 (Aether Dungeons: Pixel Legacy)
*   **현대화된 컨셉:**
    *   **고전 게임 참고:** 젤다의 전설 (초기 2D 던전 탐험), 로그 (로그라이크 요소), 디아블로 (아이템 파밍 및 성장).
    *   **현대화 요소:**
        *   **절차적 생성 던전:** 매 플레이마다 새로운 던전 구조로 지루함을 없애고 리플레이 가치를 높입니다. (Phaser Tilemap API 활용)
        *   **클래스 기반 성장 시스템:** 전사, 마법사, 궁수 등 다양한 클래스를 선택하여 각기 다른 스킬 트리와 플레이 스타일을 제공합니다.
        *   **정교한 픽셀 아트 애니메이션:** 현대적인 픽셀 아트 기법을 사용하여 캐릭터와 몬스터의 움직임을 부드럽고 다채롭게 표현합니다. (Phaser Spritesheets, Animation Manager 활용)
        *   **원소 기반 퍼즐 및 전투:** 불, 물, 번개, 땅 등의 원소 속성을 가진 몬스터와 환경 요소를 추가하여 전략적인 전투와 퍼즐 요소를 강조합니다.
        *   **Co-op 멀티플레이:** (장기적인 목표) 2인 협동 모드를 통해 던전 탐험의 재미를 더합니다.
*   **픽셀 아트 스타일 노트:**
    *   **팔레트:** NES 또는 SNES 시대의 제한된 색상 팔레트에서 영감을 받아 현대적으로 재해석한 색상을 사용합니다 (예: PICO-8 팔레트 확장).
    *   **타일셋:** 모듈형 던전 타일셋을 사용하여 다양한 환경을 표현하고 절차적 생성에 용이하게 합니다.
    *   **UI:** 미니멀리스트 픽셀 아트 UI를 사용하여 게임 세계에 몰입감을 높입니다.
*   **Phaser JS 구현 고려 사항:**
    *   **씬 관리:** 던전 층, 상점, 메인 메뉴 등을 개별 씬으로 관리하여 로드 및 전환을 효율적으로 처리합니다.
    *   **물리 엔진:** Phaser의 Arcade Physics를 사용하여 캐릭터 이동, 충돌 감지, 공격 판정 등을 구현합니다.
    *   **타일맵:** Tiled Map Editor로 생성된 JSON 타일맵을 Phaser의 Tilemap API로 로드하여 던전을 구축합니다.
    *   **UI/UX:** HTML DOM 요소를 Phaser와 연동하거나, Phaser 자체 UI 요소를 사용하여 인벤토리, 스킬 바 등을 만듭니다.

### 아이디어 2: 네온 아케이드 슈터 (Neon Arcade Shooter)

*   **게임 제목:** 사이버 제트: 네온 어썰트 (CyberJet: Neon Assault)
*   **현대화된 컨셉:**
    *   **고전 게임 참고:** 스페이스 인베이더 (기본 슈팅 메커니즘), 갤러그 (다양한 적 패턴), R-TYPE (무기 업그레이드 및 특수 능력).
    *   **현대화 요소:**
        *   **메트로이드바니아 스타일 맵:** 단방향 스크롤링 대신 탐험 요소를 가미한 넓은 맵을 제공하여 숨겨진 업그레이드와 비밀 구역을 찾게 합니다.
        *   **다양한 함선 및 무기 모듈:** 플레이어는 여러 종류의 함선을 잠금 해제하고, 다양한 무기 모듈(레이저, 미사일, 보호막 등)을 조합하여 자신만의 전략을 구사합니다.
        *   **반복 플레이를 위한 챌린지 모드:** 일일/주간 챌린지, 무한 웨이브 모드 등을 추가하여 지속적인 플레이 동기를 부여합니다.
        *   **보스 패턴 학습 및 파훼:** 각 스테이지의 보스는 고유한 공격 패턴과 약점을 가지며, 이를 학습하고 파훼하는 재미를 제공합니다.
        *   **레트로 웨이브 사운드트랙:** 신디사이저 기반의 레트로 웨이브 음악을 사용하여 게임의 분위기를 고조시킵니다.
*   **픽셀 아트 스타일 노트:**
    *   **팔레트:** 밝고 대조적인 네온 색상을 주로 사용하여 사이버펑크 분위기를 연출합니다. 어두운 배경과 대비되는 네온 효과를 강조합니다.
    *   **이펙트:** 폭발, 총알 궤적, 함선 부스터 등에 화려한 파티클 이펙트를 사용하여 시각적인 즐거움을 극대화합니다.
    *   **함선 디자인:** 최소한의 픽셀로 구성된 간결하면서도 개성 있는 함선 디자인을 추구합니다.
*   **Phaser JS 구현 고려 사항:**
    *   **충돌 감지:** Phaser의 Physics 엔진을 사용하여 정교한 충돌 감지를 구현하여 플레이어, 적, 총알 간의 상호작용을 처리합니다.
    *   **파티클 시스템:** Phaser의 파티클 시스템을 활용하여 화려한 폭발, 총알 효과, 배경 효과를 만듭니다.
    *   **오디오 관리:** 배경 음악과 다양한 효과음을 Phaser의 오디오 매니저를 통해 관리하고 재생합니다.
    *   **사용자 정의 이벤트:** 무기 업그레이드, 보스 등장, 스테이지 클리어 등 게임의 다양한 이벤트를 효율적으로 처리하기 위해 사용자 정의 이벤트를 활용합니다.
